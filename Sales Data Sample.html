📌 Title: Retail Sales Forecasting & Insightful Analysis with Python¶

🔍 Overview:¶

This project analyzes and models a retail sales dataset using pure Python to uncover business insights, track performance trends, and forecast future sales. The workflow included data cleaning, feature engineering, visual exploration, and regression modeling.

💡 Key Highlights:¶

Cleaned and prepared sales transaction data (e.g., converting date formats, handling categorical values)

Extracted features such as year, month, and product identifiers

Visualized sales trends over time using Seaborn (line plots, pie charts, etc.)

Applied Label Encoding to categorical data for modeling

Trained a XGBRegressor model to predict SALES using key features like PRICEEACH, QUANTITYORDERED, COUNTRY, and PRODUCTCODE

Evaluated model performance using R-squared and visualized predictions

🎯 Business Impact:¶

This project demonstrates how retailers can use Python to explore, interpret, and predict sales data to inform stock decisions, optimize pricing, and better understand customer demand patterns.

le = LabelEncoder()
