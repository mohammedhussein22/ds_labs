Project Title: Sales Forecasting and Dynamic Pricing Simulation¶

Description:¶

In this project, I built a system to help retailers predict demand and optimize prices using machine learning and time series analysis. First, I used time series forecasting (ARIMA) to predict future product demand based on historical sales. This helps stores prepare inventory and avoid stockouts. Then, I developed a dynamic pricing simulation using a regression model. It predicts how changes in product price affect sales. I tested different prices and calculated how much revenue each price would generate. This helps find the best price to maximize revenue.

Tools & Technologies:¶

Python, Pandas, NumPy

scikit-learn (for pricing model)

statsmodels (for ARIMA time series)

Matplotlib / Seaborn (for plots)

Key Features:¶

Forecasted product demand using ARIMA Built a machine learning model to predict sales based on price, discounts, and promotions Simulated different pricing scenarios to find the most profitable price

from statsmodels.graphics.tsaplots import plot_acf, plot_pacf
from statsmodels.tsa.statespace.sarimax import SARIMAX

product_id = 'P0005'
product_df = df[df['Product ID'] == product_id].copy()
product_df['Date'] = pd.to_datetime(product_df['Date'])

# Step 2: Aggregate daily demand
daily_sales = product_df.groupby('Date')['Demand Forecast'].sum().asfreq('D')  # Ensure consistent daily frequency

# Step 3: Train-test split
train = daily_sales[:-30]
test = daily_sales[-30:]

# Step 4: ACF and PACF plots to analyze lag structure
plt.figure(figsize=(12, 4))
plot_acf(train.dropna(), lags=30)
plt.title('ACF Plot')
plt.show()

plt.figure(figsize=(12, 4))
plot_pacf(train.dropna(), lags=30)
plt.title('PACF Plot')
plt.show()

# Step 5: Manually set ARIMA order (change as needed)
manual_order = (0,1,1)  # Example: ARIMA(2,1,2)

# Step 6: Fit the SARIMAX mode)l
sarimax_model = SARIMAX(train, order=manual_order)
sarimax_result = sarimax_model.fit(disp=False)

# Step 7: Forecast next 30 days
forecast = sarimax_result.forecast(steps=30)

# Step 8: Plot forecast vs. actual
plt.figure(figsize=(14, 6))
plt.plot(train.index, train, label='Train')
plt.plot(test.index, test, label='Actual', color='green')
plt.plot(forecast.index, forecast, label='Forecast', color='red')
plt.title(f'Manual ARIMA Forecast for Product {product_id}')
plt.xlabel('Date')
plt.ylabel('Units Sold')
plt.legend()
plt.grid(True)
plt.tight_layout()
plt.show()

# 1. Use a single test sample
sample = x_test.iloc[0].copy()

# 2. Define a price range around the current price
price_range = np.linspace(sample['Price'] * 0.8, sample['Price'] * 1.2, 10)

sim_results = []

# 3. Loop through price points and simulate revenue
for price in price_range:
    temp = sample.copy()
    temp['Price'] = price
    
    # Ensure input is 2D array for model.predict
    predicted_units = model.predict([temp])[0]
    revenue = price * predicted_units
    
    sim_results.append({
        'Price': price,
        'Predicted Units': predicted_units,
        'Revenue': revenue
    })

# 4. Convert results to DataFrame
sim_df = pd.DataFrame(sim_results)

# 5. Plot Revenue vs. Price
plt.figure(figsize=(10, 5))
plt.plot(sim_df['Price'], sim_df['Revenue'], marker='o', color='blue')
plt.title('Revenue vs. Price Simulation')
plt.xlabel('Price')
plt.ylabel('Simulated Revenue')
plt.grid(True)
plt.tight_layout()
plt.show()
